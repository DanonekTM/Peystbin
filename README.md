# :clipboard: Peystbin
A Text Storage Site.

[Bugs](https://github.com/DanonekTM/Peystbin/issues)

## Features
* Clean URL's.
* Account registration.
* Light & Dark theme.
* Public & Private pastes.
* Anonymous pastes.
* Code highlighting.
* Raw view.
* Word wrapping.
* Paste editing & deleting.
* "Free" & "Premium" accounts.
* Password change.
* Login history.
* Password recovery via email.
* Option to delete & upgrade account.

## Prerequisites
* PHP 5.6+
* MYSQL server.
* Web server.
