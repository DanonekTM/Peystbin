# :clipboard: Peystbin

* To begin, install PHP, MYSQL, and a web server of your choice on your system.  
* After you have done installing the prerequisites put the files in the root directory of your web server.

* When you're done with that, create a new database on your server and run the **pastebin.sql** file inside of it.  
* Edit the database config file in **application/config/settings.php** with your database credentials and modify the environment.  
* Following, edit **application/config/config.php** for more settings related to your web server, email, etc.

You are finished and ready to go!
